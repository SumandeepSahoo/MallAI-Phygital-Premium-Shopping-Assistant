# MallAI Simulation Model

This folder contains a Python simulation that demonstrates how online orders
are batched and fulfilled within a 1–2 hour SLA using the proposed MallAI system.

## Files
- simulation_model.py : Main Python script to run the simulation.
- requirements.txt : Required Python libraries.
- output.csv : Sample output generated by the simulation.

## Logic
1. Generate 100 random online orders with random arrival times.
2. Group orders into picking batches every 30 minutes.
3. Assign picking and delivery times, then calculate SLA compliance.
4. Export results as a CSV file and display summary metrics.

## How to Run
1. Install Python 3.9+ and required libraries:
   pip install -r requirements.txt
2. Run the script:
   python simulation_model.py
3. View output:
   Check output.csv or printed metrics in the terminal.

## Dependencies
- pandas

## Expected Output
- A CSV with columns: OrderID, ArrivalTime, BatchTime, PickTime(mins), DeliveryTime(mins), SLA_Met
- Summary of average pick time and SLA compliance rate.
